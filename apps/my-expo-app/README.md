## Get started

1. Install dependencies
   ```bash
   pnpm i -r
   ```
2. Start the app - this app runs separately from the `pnpm run dev:all` command

   ```bash
   pnpm run dev:all #for the servers
   cd apps/my-expo-app
   pnpm run start
   ```
